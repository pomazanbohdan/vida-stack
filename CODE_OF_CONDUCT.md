# Code of Conduct

Status: canonical

Purpose: define the public participation and conduct expectations for the Vida Stack repository.

## Expected Behavior

Participants in this repository are expected to:

1. communicate respectfully and directly,
2. critique ideas, designs, and implementation decisions without personal attacks,
3. keep discussion grounded in evidence, reproducible behavior, or clearly labeled proposals,
4. avoid harassment, discrimination, intimidation, or hostile conduct,
5. respect maintainer boundaries on scope, licensing, and contribution posture.

## Unacceptable Behavior

The following behavior is not acceptable in repository discussions, issues, pull requests, or other project surfaces:

1. harassment or abusive language,
2. personal attacks or repeated hostility,
3. discriminatory or exclusionary behavior,
4. deliberate disruption, spam, or bad-faith escalation,
5. publication of private, sensitive, or security-relevant information without permission.

## Enforcement

Repository maintainers may:

1. moderate, edit, or remove content,
2. close discussions or issues,
3. restrict participation,
4. escalate severe cases through the repository security/support surfaces.

Enforcement decisions are project-owned and may be taken without preserving a public debate trail in the affected thread.

## Reporting

For conduct concerns:

1. use the support surface in [SUPPORT.md](SUPPORT.md) for general guidance,
2. use [SECURITY.md](SECURITY.md) when the report intersects with vulnerability or private disclosure concerns.
