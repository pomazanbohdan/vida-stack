# Support

Status: canonical

Purpose: define the public support and help surface for the Vida Stack repository.

## Use This Surface For

1. questions about repository usage,
2. clarification on install and upgrade behavior,
3. help interpreting repository docs and product specs,
4. direction on whether something is a bug, discussion topic, or feature request.

## Support Channels

Use repository issues for:

1. reproducible bugs,
2. documentation defects,
3. bounded repository problems with clear expected behavior.

Use repository discussions or issue-style planning threads, when available, for:

1. design questions,
2. roadmap discussion,
3. specification clarification,
4. higher-level product or framework direction questions.

Use [SECURITY.md](SECURITY.md) instead of public channels for:

1. vulnerabilities,
2. secrets exposure,
3. exploit or abuse reports.

## Support Boundaries

This repository does not guarantee:

1. SLA-backed support,
2. immediate turnaround,
3. acceptance of every feature request or integration ask.

Support remains bounded by the current repository roadmap and maintainer capacity.
